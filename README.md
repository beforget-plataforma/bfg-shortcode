# BFG Shortcode

Plugin para Wordpress con un listado de funciones que obtienen distinto contenido de la plataforma.

### Listado de shortcode para implementar:

Eventos de calendario
```
[events-shortcode]
```

```
[members-shortcode]
```

```
[post-shortcode]
```

```
[sesiones-shortcode]
```
